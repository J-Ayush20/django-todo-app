# Django Todo App

A simple and responsive Todo application built using Django.

## Features

* Create new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Add task descriptions
* Set due dates
* Simple and clean user interface

## Tech Stack

* Python 3
* Django
* SQLite
* HTML
* CSS

## Installation

1. Clone the repository

```bash
git clone https://github.com/J-Ayush20/django-todo-app.git
```

2. Navigate to the project directory

```bash
cd django-todo-app
```

3. Create and activate a virtual environment

```bash
python -m venv env
env\Scripts\activate
```

4. Install dependencies

```bash
pip install django
```

5. Apply migrations

```bash
python manage.py migrate
```

6. Run the development server

```bash
python manage.py runserver
```

7. Open your browser and visit

```text
http://127.0.0.1:8000/
```
